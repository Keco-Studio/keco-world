# Garen一脉纪事

Garen是这一脉的始祖，其血脉历经59代的繁衍。如今这一脉仍在延续。

## 始祖

Garen信奉：『见狼要跑，有墙要躲』，时在第2年夏。 Garen死于严寒，时在第2年冬。

## 第2代

Odo诞生于第2年夏，父母是Garen与族外的伴侣。 Col诞生于第2年冬，父母是Garen与族外的伴侣。 Col信奉：『能摘就摘，莫要迟疑』，时在第2年冬。 Col死于严寒，时在第2年冬。 Odo信奉：『严冬几乎夺走我的命』，时在第4年夏。

## 第3代

Bram诞生于第3年夏，父母是Odo与族外的伴侣。 Wren诞生于第3年冬，父母是Odo与族外的伴侣。 Bram信奉：『寒冬近乎致命，我记得』，时在第4年夏。 Wren信奉：『寒冬近乎致命，我记得』，时在第4年夏。 Wren死于饥饿，时在第4年夏。

## 第4代

Quinn诞生于第4年夏，父母是Bram与族外的伴侣。 Runa诞生于第4年夏，父母是Odo与族外的伴侣。 Fenna诞生于第4年冬，父母是Odo与族外的伴侣。 Sorrel诞生于第5年夏，父母是Bram与族外的伴侣。 Petra诞生于第5年夏，父母是Bram与族外的伴侣。

## 结语

历经数代演变，这一脉比先祖更热衷于采集，更疏于亲近同伴，更疏于远行。
