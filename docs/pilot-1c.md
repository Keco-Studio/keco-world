## 2. 三个重量级发现

### 2.1 50k tick 恰好卡在 50 代边界（待拍板 #6 的实证答案）

Evolutionary/NoCulture 全部 24 个 seed 全部存活，maxGen 落在 47–52——代际速率约 1 代/1000 tick。S1 的 maxGen ≥ 50 在 50k tick 下是掷硬币。**建议**：官方运行延长到 60k tick（预计 gen 58–62，余量充足；成本可忽略——实测约 45 秒/seed）。

### 2.2 Fixed 臂会灭绝——S1 的"任一失败即 Stop"需要按臂域定

Fixed 臂 3/12 seed 中途灭绝（maxGen 18/32/42 处）；这与 baseline-arms.md 记录的 clone 遗传血脉聚结（livingLineages→1）一致：单血脉人口没有多样性缓冲，一次坏冬天就清零。Handcrafted（同为 clone，但设计参数更稳）12/12 存活。**这是比较性结果（进化提高存续），不是"世界不稳定"**——但 §6 的 Stop 规则写的是"S 组任一失败"，会把基线臂的死亡误判为全局中止条件。**建议**：S1 作为中止判据只约束 Evolutionary 臂；基线臂的存活率降级为报告项（它本来就是三轴里"生存稳定性"要对照的东西）。

### 2.3 观看无聊威胁第三次升级：50 代世界的慢性 idle（S3 大面积失败）

退化检查（15k tick）看到 idle 份额爬到 ~50% 平台 + 瞬时越线；50k tick 下变成**慢性**：Evolutionary 最长 25 个连续 chunk（25,000 tick！）idle>600‰，NoCulture 19 连块，Fixed 也有 6 连块。越线段远超季节长度（400 tick），不是冬季蛰伏。唯一干净通过 S3 的是 **Handcrafted（12/12）**——设计出来的人格参数让世界一直动。

诚实的解读顺序：这**首先是产品威胁的实证**（§15"生存最优但观看无聊"——进化收敛到的稳态确实是低活动性的），**其次才是阈值校准问题**。任何把 S3 放宽到试跑能过的改法，都必须同时在报告里承认"50 代进化世界的默认观感是安静的"。可选的应对（冻结前择一或组合）：(a) S3 保持现状，接受 Evolutionary 大概率 FAIL S3 并按 Iterate 处理（诚实但基本预定结论）；(b) 把 S3 改为"idle 份额长期均值 < 700‰"之类的更宽判据，越线记录降级为报告项——同时把"观看无聊"列入 1B 产品化必须解决的设计债；(c) 视 S3 为世界设计（manifest 参数/行为系统）在 1C 前需要迭代的信号——但这会推迟冻结。**试跑数据倾向 (b)+设计债记录**：世界没有死（S1 之外全指标健康），它只是变得安静。

### 2.4（附带）文化层归因的预告

NoCulture 臂在 N1（11/12）和 N2（10/12）上与 Evolutionary（12/12、10/12）几乎同水平；平均 N1 漂移 0.475 vs 0.543。按 §3 预注册的归因规则，正式运行若复现此格局，报告必须如实声明：**行为漂移与多样性维持主要由可遗传结构驱动，信念累积的增量贡献有限**（本试跑口径下 ~+0.07 verbL1）。文化层对主终点（传记可看性——信念句是叙事素材）的贡献与此正交，仍由人评检验。

## 3. 仪器状态

- 全管线一次通过；`analyze` 的 Stop 建议行为符合 §6 字面规则（正确地被 S1/S3 触发——这正是上面两条域定/校准讨论的由来）。
